\section{Bivariate data} \subsection{Preliminaries} \begin{framed} \begin{verbatim} import scipy import numpy as np import matplotlib at plt \end{verbatim} \end{framed} \subsection{What is Bivariate and Multivariate data?} A dataset with two variables contains what is called \textbf{bivariate data}. For example, the heights and weights of people (e.g. for the purposes of determining the extent to which taller people weigh more). Common bivariate statistical analyses include \begin{itemize} \item Correlation Analysis \item Simple Linear Regression \end{itemize} Multivariate Data describes the situation where a phenomenum is described by two or more variables. For example, Height Weight Body Mass Index, Choloestorol level etc \subsection{Scatter Plot} A scatter plot of two variables shows the values of one variable on the Y axis and the values of the other variable on the X axis. Scatter plots are well suited for revealing the relationship between two variables.
