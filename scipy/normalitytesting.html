\section*{scipy - Testing Normality} \begin{itemize} \item \texttt{normaltest} returns a 2-tuple of the chi-squared statistic, and the associated p-value. \item Given the null hypothesis that x came from a normal distribution, the p-value represents the probability that a chi-squared statistic that large (or larger) would be seen. \item If the p-val is very small, it means it is unlikely that the data came from a normal distribution. \item Traditionally, in statistics, you need a p-value of less than 0.05 to reject the null hypothesis. \item For this example, we would reject the null hypothesis. \end{itemize} \begin{framed} \begin{verbatim} y = stats.uniform.rvs(size = 100) stats.normaltest(y) \end{verbatim} \end{framed} \begin{verbatim} # Output # Test Statistics, P-value (31.487039026711866, 1.4543748291516241e-07) \end{verbatim} %========================================================== % \subsection*{Exercise} \noindent Perform the normality test for datasets x1 and x2. \begin{framed} \begin{verbatim} import scipy.stats as stats x1 = stats.norm.rvs(size = 100) x2 = stats.norm.rvs(size = 20) stats.normaltest(x1) stats.normaltest(x2) \end{verbatim} \end{framed}
