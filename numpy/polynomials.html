### *{Polynomial Mathematics} NumPy supplies methods for working with polynomials. Given a set of roots, it is possible to show the polynomial coefficients: \begin{framed} \begin{verbatim} >>> np.poly([-1, 1, 1, 10]) array([ 1, -11, 9, 11, -10]) \end{verbatim}

\noindent Here, the return array gives the coefficients corresponding to \[(x+1)(x-1)(x-1)(x-10) = x^4 -11x^2 + 9x^2 +11x-10 \] . \noindent The opposite operation can be performed: given a set of coefficients, the root function returns all of the polynomial roots: \begin{framed} \begin{verbatim} >>> np.roots([1, 4, -2, 3]) array([-4.57974010+0.j , 0.28987005+0.75566815j, 0.28987005-0.75566815j]) \end{verbatim}

\noindent Notice here that two of the roots of \[x^3 + 4x -2x +3\] are complex numbers (real and imaginary components). %----------------------------------------------- %

### *{Integration} Coefficient arrays of polynomials can be integrated. Consider integrating \[ x^3 + x^2 + x +1\] to \[ \frac{x^4}{4} + \frac{x^3}{3} + \frac{x^2}{2} + x + C \] \begin{framed} \begin{verbatim} >>> # By default, the constant is set to zero: >>> np.polyint([1, 1, 1, 1]) array([ 0.25 , 0.33333333, 0.5 , 1. , 0. ]) \end{verbatim}

\noindent Similarly, derivatives can be taken. Reversing the previous example, we get \begin{framed} \begin{verbatim} >>> np.polyder([1./4., 1./3., 1./2., 1., 0.]) array([ 1., 1., 1., 1.]) \end{verbatim}

%----------------------------------------------- % The functions \texttt{polyadd}, \texttt{polysub}, \texttt{polymul}, and \texttt{polydiv} also handle proper addition, subtraction, multiplication, and division of polynomial coefficients, respectively. The function polyval evaluates a polynomial at a particular point. Consider \[ x^3 - 2x^2 + 2 \] evaluated at $x=4$. \begin{framed} \begin{verbatim} >>> np.polyval([1, -2, 0, 2], 4) 34 \end{verbatim}

%----------------------------------------------- % ### *{Line Fitting} Finally, the \texttt{polyfit} function can be used to fit a polynomial of specified order to a set of data using a least-squares approach: \begin{framed} \begin{verbatim} >>> x = [1, 2, 3, 4, 5, 6, 7, 8] >>> y = [0, 2, 1, 3, 7, 10, 11, 19] >>> np.polyfit(x, y, 2) array([ 0.375 , -0.88690476, 1.05357143]) \end{verbatim}

\noindent The return value is a set of polynomial coefficients. More sophisticated interpolation routines can be found in the SciPy package.
